# 更新指南

## 最簡單的更新方式

1. 進入 GitHub repository。
2. 點要修改的檔案，例如 `index.html`。
3. 點右上角鉛筆圖示編輯。
4. 修改完成後按 `Commit changes`。
5. Vercel 會自動重新部署。

## 上傳新圖片

1. 進入 `assets/` 資料夾。
2. 點 `Add file`。
3. 選 `Upload files`。
4. 上傳圖片。
5. Commit changes。
6. 到 `index.html` 修改圖片檔名。

## 建議檔名規則

請使用英文小寫與連字號，避免中文檔名造成路徑問題。

範例：

- `profile.jpg`
- `family-photo.jpg`
- `stool-color-card.jpg`
- `jaundice-guide.jpg`
